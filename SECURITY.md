# Security Policy

## Supported Versions

Security fixes are applied to the latest code on the default branch.

## Reporting a Vulnerability

1. Do not open public issues for suspected vulnerabilities.
2. Report privately via GitHub Security Advisories:
   - https://github.com/shuto-S/codex-app-mobile/security/advisories/new
3. Include impact, reproduction steps, and affected files/components.
4. After confirmation, fixes will be prepared and disclosed with an appropriate timeline.
